# PawsStore

Полностью переписанный проект интернет-магазина по ТЗ с поэтапной историей коммитов.

## Что реализовано

- Сквозной `Header`:
  - Логотип со ссылкой на каталог
  - Навигация
  - Иконка корзины с `badge` количества товаров (обновляется в реальном времени)
- Каталог:
  - Сетка карточек товаров
  - Сортировка: `Name (A-Z)`, `Name (Z-A)`, `Price (Low-High)`, `Price (High-Low)`
  - Фильтрация: рейтинг (`5+`, `4+`, `3+`) и диапазон цены (`min/max`)
  - Клик по названию/картинке ведет на страницу товара
  - Кнопка добавления в корзину прямо из карточки
  - Анимация карточки в стиле референса (плавающая информационная панель при hover)
- Карточка товара:
  - Галерея из 3+ изображений, переключение миниатюрами и кнопками `prev/next`
  - Блок данных: название, цена, рейтинг, количество, кнопка добавления в корзину
  - Аккордеон: технические характеристики и расширенное описание
  - Динамика кнопки: `Already in Cart` если товар уже в корзине
- Корзина:
  - Список товаров с названием, подзаголовком и ценами
  - Изменение количества `+ / -`
  - Удаление позиции
  - Мгновенный пересчет суммы
  - Промокод `SAVE10` (скидка 10%)
  - Ошибки валидации: пустой ввод / неверный код
  - `Empty State` с кнопкой возврата в каталог
- Persistence:
  - Корзина и промокод сохраняются в `localStorage`
- Адаптив:
  - Интерфейс покрывает диапазон от `375px` до `1440px`

## Технологии

- JavaScript (ES6+, Vanilla)
- HTML5
- CSS3
- Hash-routing (`#/catalog`, `#/product/:id`, `#/cart`)

## Структура проекта

- `index.html` — корневой шаблон
- `src/main.js` — вход приложения
- `src/router.js` — маршрутизация и глобальные события
- `src/data/products.js` — данные каталога
- `src/state/store.js` — состояние и localStorage
- `src/pages/*.js` — страницы (`catalog`, `product`, `cart`)
- `src/ui/*.js` — shell, карточки, toast, DOM refs
- `src/styles/*.css` — слои стилей (base/layout/components/pages/responsive)

## Локальный запуск

Открыть `index.html` напрямую

или через простой локальный сервер:

```bash
python -m http.server 5173
```

После запуска открыть: `http://localhost:5173`
